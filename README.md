### Ruby XDCC getter (name subject to change)

###### Required
- Ruby
- cinch (`gem install cinch`)

###### Usage

`ruby xdcc.rb <server> <bot_name> <file_number> [-c <channel_name>] [-d <download_folder>] [-o <bot_log_file>]`
